To run the program-

Step 1: Install Anaconda
Step 2: Open Anaconda Promt
Step 3: Install all the required libraries- NumPy, dlib, cv2, imutils
Step 4: Change directory to the folder containing "people_counter.py"
Step 5: Run the command- "python people_counter.py --prototxt mobilenet_ssd/MobileNetSSD_deploy.prototxt --model mobilenet_ssd/MobileNetSSD_deploy.caffemodel --input videos/video1.mp4 --output output/output1.avi"
